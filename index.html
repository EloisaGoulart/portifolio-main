Tux Games

Tux Games é um catálogo web desenvolvido em HTML, CSS e JavaScript, com um banco de dados usado para armazenar as mensagens enviadas pelo formulário de contato. O projeto é dividido em três páginas (vitrine, catálogo e contato) cada uma com seus próprios arquivos de estrutura, estilo e script.

No catálogo, o JavaScript carrega os jogos, cria os cards e aplica o sistema de filtros, organizando os itens por categoria a partir da manipulação do DOM. Já a página de contato envia os dados preenchidos pelo usuário para o banco de dados após validações básicas.

A estrutura do código separa HTML, CSS e JavaScript por função, com o JS responsável pelas funcionalidades do catálogo e do envio das mensagens.

Portal SESI em Casa é uma plataforma estudantil desenvolvida com Next.js e TypeScript, usando Supabase como backend para autenticação, banco de dados, armazenamento de arquivos e segurança. O sistema conta com perfis separados para alunos, professores, responsáveis e administrador, cada um com permissões definidas diretamente no Postgres.

As regras de segurança foram criadas com RLS (Row Level Security), garantindo que cada usuário acesse apenas seus próprios dados. Professores visualizam e gerenciam apenas suas turmas, responsáveis têm acesso somente aos alunos vinculados e alunos veem exclusivamente suas informações. Essas políticas foram aplicadas nas tabelas de notas, frequência, comunicados, materiais e turmas.

A plataforma inclui CRUD completo de usuários e turmas, upload de arquivos via Supabase Storage, dashboards de desempenho, calendário escolar, comunicados e gerenciamento de avaliações e frequência.

Hecho Base - Detalhes

Assistir Vídeo do Projeto

Hecho é um aplicativo mobile desenvolvido em React Native (Expo) com TypeScript, utilizando Supabase para autenticação, banco de dados e armazenamento de imagens.

O app conta com CRUD completo de perfis, suporte a múltiplas contas, upload de fotos via câmera, favoritos por usuário, gerenciamento de sessão e sincronização em tempo real com Postgres.

Também foram criadas APIs próprias para cálculo de signo, leitura de tarot e geração de dados místicos, unindo lógica de datas, tabelas personalizadas e manipulação de JSON diretamente no React Native.

PedalCrew

Ver Projeto no Behance

O PedalCrew é um protótipo pensado para ciclistas, criado para organizar treinos, acompanhar rotas e permitir interação entre usuários. A plataforma inclui rotas personalizadas, monitoramento de desempenho, feed social, perfis com publicações, grupos e comunidades, além de chat para comunicação entre ciclistas e recursos de apoio como integração com wearables e modo SOS.

O desenvolvimento seguiu os passos do Design Thinking, começando com pesquisa de usuários, análise de concorrentes e definição de personas. As telas foram prototipadas em Figma e Adobe XD, priorizando fluxos claros, navegação simples e organização de informações, sempre pensando na experiência de quem pedala, seja para lazer ou prática esportiva.
